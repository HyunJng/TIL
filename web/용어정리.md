# 웹개발 하며 많이 듣는 용어 정리

## JS SDK(JavaScript Software Development Kit)
* 웹 개발자가 JavaScript 사용해서 특정 서비스를 쉽게 연동하고 기능을 구현할 수 있도록 
  제공되는 라이버르리, API 등을 모아둔 개발 키트
* `<script>` 태그로 호출되는데 이 태그는 브라우저 모델 상 CORS 검사 대상이 아니다
* JS는 로드되면 현재 페이지의 origin 권한으로 실행된다.
* JS SDK 자체가 수많은 외부 사이트에서 실행되도록 구현된거고, 우리의 API를 호출하므로
  CORS 정책을 반드시 고려해야한다.

---
## CDN(Content Delivery Network)
전 세계에 분산된 서버에 정적 리소스를 복제해두고
사용자와 가장 가까운 서버에서 빠르게 내려주는 시스템
* 보통 정적 리소스를 올려둔다. (JS SDK는 CDN에 올려둠)

---
## AJAX(Asynchronous JS And XML)
* 페이지 전체 새로고침 없이 JS가 백엔드랑 통신해서 일부 UI만 갱신하는 "방식". 객체나 함수이름 아님
### XHR(XMLHttpRequest)
* 브라우저가 제공하는 HTTP 통신 객체
### fetch
* XHR 문제를 해결하려고 나온 현대 표준 API
