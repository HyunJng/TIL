### 비동기 작업

자바스크립트는 **싱글스레드 언어** 이지만, WebBrowser의 Web APIs를 이용해 멀티스레드처럼 동작한다.

1. Web APIs에게 타이머와 함께 콜백함수를 넘겨줘서
2. 타이머가 끝나면 콜백함수를 돌려받아 실행


### Promise
비동기 작업을 감싸는 객체
* 상태: 대기--resolve---->성공, 대기 ---reject--->실패
```javascript
const promise = new Promise((resolve, reject) => {
    // 비동기 작업 실행하는 함수 : executor
    setTimeout(() => {
        const num = 10;
        if (typeof num === "number") {
            resolve(num + 10);
        } else {
            reject("num은 숫자가 아닙니다");
        }
    }, 2000);
})
    // then: 성공 후에 
    .then((value) => (console.log(value)))
    // catch: 실패 후에 
    .catch((error) => (console.log(error)));
```

### async & await
* async
  *  어떤 함수를 비동기 함수로 만들어주는 키워드
  *  함수가 프로미스를 반환하도록 변환해주는 키워드
* await
  * async 함수 내부에서만 사용 가능한 메서드
  * 비동기 함수가 다 처리되기까지 기다리는 역할
```javascript
async function getData() {
    return {
        name: "홍길동"
    };
} 

async function printData() {
    const data = await getData();
    console.log(data);
}
```
